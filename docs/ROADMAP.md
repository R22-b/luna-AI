# 🗺️ Luna AI Roadmap

This document outlines the future direction of Luna AI. Features are prioritized based on community feedback and development capacity.

---

## 📊 Current Status: v2.0.0 (Stable)

**Released:** June 28, 2026

**Features:**
- ✅ 27-Brain AI Cascade
- ✅ Full PC Control (Windows)
- ✅ Project Architect & Guardian
- ✅ Voice I/O (TTS + STT)
- ✅ Document Generation (Word, PDF, PPT)
- ✅ Self-Evolution Engine
- ✅ SQLite Memory System
- ✅ Plugin System
- ✅ 5-Layer Security

## 🚧 Known Limitations

### Current (v2.0.0)

- **Windows Only** - macOS/Linux coming soon
- **Single User** - No multi-user support yet
- **Local Storage** - Cloud sync optional in future
- **Limited Mobile** - No mobile app yet
- **No Fine-tuning** - Can't train on user data

### Planned Fixes

All limitations above are planned for future releases.
