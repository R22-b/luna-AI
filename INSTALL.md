# 💾 Installing Luna AI 2.0

## Quick Start
1. Download `Luna_AI_2.0_Release.zip` and extract the files.
2. If you want a permanent installation, double-click **`Luna AI Setup 2.0.0.exe`**.
3. If you want to run Luna from a USB drive without installing, use **`Luna AI 2.0.0.exe`** (Portable).

## First Boot
- Windows SmartScreen may present a blue warning box since this app isn't digitally signed yet. Click **More Info -> Run Anyway**.
- Upon opening, navigate to the **Settings (⚙️)** gear icon in the bottom right corner.
- Enter your **Groq API Key** and **Gemini API Key**.
- Say "Hey Luna" to test the microphone!

## System Requirements
- OS: Windows 10 / 11
- Memory: 4GB RAM minimum (Luna uses ~300MB idle).
- Disk Space: 500MB free.
- Dependencies: `Node.js` and `npm` installed (required for autonomous coding tasks).
